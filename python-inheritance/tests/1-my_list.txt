Test suite for the MyList class defined in my_list.py.

This test file is designed to be run using Python's doctest module:
$ python3 -m doctest test_mylist.txt

Testing MyList.print_sorted(self) method:

from my_list import MyList

1. Test case with positive integers

test1 = MyList()
test1.append(5)
test1.append(1)
test1.append(3)
print(test1)
[5, 1, 3]
test1.print_sorted()
[1, 3, 5]

Verify the original list remains unmodified (crucial test)

print(test1)
[5, 1, 3]

2. Test case with duplicates and negative numbers

test2 = MyList()
test2.extend([10, -5, 0, 10, -1])
print(test2)
[10, -5, 0, 10, -1]
test2.print_sorted()
[-5, -1, 0, 10, 10]

Verify the original list remains unmodified

print(test2)
[10, -5, 0, 10, -1]

3. Test case with an empty list

test3 = MyList()
print(test3)
[]
test3.print_sorted()
[]
print(test3)
[]
